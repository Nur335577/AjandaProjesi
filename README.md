# AjandaProjesi
Bu proje, kişisel ajanda uygulaması olarak geliştirilmiştir.
Kullanıcıların günlük notlarını, etkinliklerini ve hatırlatmalarını kaydedip yönetmesini sağlar.

# Özellikler 
- Etkinlik ekleme, silme ve güncelleme
 - SQL veritabanı ile veri saklama
- C# Windows Forms arayüzü
 - Tarih ve saat seçimi ile hatırlatma sistemi
